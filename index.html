Problem overview.

The system is developed for the purpose of master thesis project. The main focus of the solution is detection of temporal changes in large-scale scenes reconstructed with use of Structure from Motion(SfM) techniques. Given the SfM reconstruction and new images of the scene, system automatically determines position of new cameras and applies image-based change detection algorithms to produce change mask. Change mask is then projected into 3D space using available from SfM process cameras parameters in order to indicate areas in the scene that might change.

Dependencies

The code is written solely in C++ however it leverages several external libraries and software. In order to run the code one has to make sure first to have installed following components:

  • PCL libraries
  • VCG libraries
  • OpenCV
  • boost libraries
  • VisualSFM(command line functionality is required - only CUDA enabled hardware)
