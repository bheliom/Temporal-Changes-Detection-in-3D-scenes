Project overview.

Project is developed in frame of master thesis project. The main focus of the project is detection of temporal changes in large-scale scenes reconstructed with use of Structure from Motion techniques.

Dependencies

The code is written solely in C++ however it leverages several external libraries and software. In order to run the code one has to make sure first to have installed following components: -PCL libraries -VCG libraries -OpenCV -boost -VisualSFM(command line functionality is required - only CUDA enabled hardware)
